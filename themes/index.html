Beautiful, production-ready themes for your HtmlBuilder projects

How to Use Themes

  1. Browse the available themes below
  2. Choose a theme that fits your project style
  3. Copy the code example from your chosen theme
  4. Use it in your HtmlBuilder code with builder.useTheme(Theme.THEME_NAME)
  5. Build your HTML and enjoy your beautifully styled page

Want to Contribute Your Own Theme?

  1. Fork the HtmlForge repository
  2. Create a new folder in themes/YourThemeName/
  3. Add your style.css file with your custom styles
  4. Update the themes.json file with your theme's CSS and metadata
  5. Add your theme to the Theme enum in Java
  6. Submit a pull request with your amazing theme!

Note: Themes are now fetched dynamically from the JSON file, so users can access them without downloading the repository!
